# CristalKilakila：小水晶的苔藓园艺手记

部分观察笔记已储存在 `/docs/` 与 `/poems/` 文件夹，建议黄昏后查阅。

## 🌕 风未止，苔仍绿

这个花园不会解释自己。  
你若懂，就来散步。  
你若不懂，那请以为它只是一个植物笔记。

在无雨的夜里，有人轻声说：

> “锚，仍在。”
